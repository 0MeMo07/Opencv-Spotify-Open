# Opencv-Spotify-Open

This code allows you to open Spotify with your hands following using the Mediapipe and OpenCV libraries.

## Requirements

- Python 3.x
- cv2 (OpenCV)
- mediapipe
- webbrowser

Make sure you have installed the above requirements.
